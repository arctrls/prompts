# Explore Topic - $ARGUMENTS

Obsidian vault에서 특정 주제에 관련된 문서를 수집하고 태그별로 그룹화합니다.

## 필수 규칙

- 모든 파일 경로는 **절대경로**로 출력하세요.

## 작업 프로세스

1. `collect-topic` 도구를 호출합니다:
   - `topic`: "$ARGUMENTS"
   - `topK`: 50
   - `similarityThreshold`: 0.3
2. 수집된 문서를 태그 그룹별로 정리합니다.
3. 관련 주제 제안을 포함하여 보고서를 작성합니다.

## 출력 형식

```
## 주제 탐색: "$ARGUMENTS"

### 통계
- 수집된 문서: N건
- 총 단어 수: N
- 관련 태그: N개

### 태그별 문서 그룹

#### #tag/category1 (N건)
1. **문서 제목** - 경로, 단어 수
2. ...

#### #tag/category2 (N건)
1. ...

### 관련 주제 제안
- 주제1 (연관 문서 N건)
- 주제2 (연관 문서 N건)

### 활용 제안
- 이 주제로 작성할 수 있는 새 문서 아이디어
- 보강이 필요한 영역
```
