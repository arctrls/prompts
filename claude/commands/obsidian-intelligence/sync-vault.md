# Sync Vault

Obsidian vault의 모든 마크다운 문서를 스캔하여 벡터 스토어에 임베딩합니다.
검색, 태깅, 관련 문서, 중복 탐지 등 시맨틱 기능을 사용하기 전에 반드시 실행해야 합니다.

## 작업 프로세스

1. `sync-vault` 도구를 호출합니다.
2. 동기화 결과를 아래 형식으로 보고합니다.

## 출력 형식

```
## Vault 동기화 완료

- 총 파일: N개
- 처리 완료: N개
- 건너뜀 (변경 없음): N개
- 실패: N개
```
