# Vault Health Report

Obsidian vault의 전체 상태를 분석하고 중복 문서를 탐지하여 종합 보고서를 작성합니다.

## 필수 규칙

- 모든 파일 경로는 **절대경로**로 출력하세요.

## 작업 프로세스

1. `analyze-vault` 도구를 호출하여 vault 통계를 수집합니다.
2. `detect-duplicates` 도구를 호출하여 중복 문서를 탐지합니다.
3. 수집된 데이터를 아래 형식으로 종합 보고서를 작성합니다.

## 보고서 형식

```
## Vault 통계 요약
- 총 파일 수 / 총 단어 수
- 확장자별 파일 분포
- 디렉토리별 파일 분포 (상위 10개)

## 태그 분석
- 태그 빈도 상위 20개
- 미태깅 파일 목록 (있으면)
- 태그 카테고리별 분포

## 중복 문서 탐지
- 중복 그룹 수
- 각 그룹: 마스터 문서 + 중복 문서 목록 + 유사도 점수
- 정리 권장 사항 (있으면)

## 개선 제안
- vault 구조 개선 포인트
- 태깅 품질 향상 제안
```
